KEY FEATURES
================================================================================

1. INTELLIGENT DELTA DETECTION
   • Snapshots capture: transforms, geometry, materials, lights, cameras
   • Hashing avoids floating-point noise
   • Change detection via hash comparison

2. AUTOMATIC FALLBACK
   • Delta > 1MB → Full scene transfer
   • Geometry changes → Server requests full sync
   • Material changes → Server requests full sync
   • Always consistent, never corrupted

3. PERFORMANCE METRICS
   • Deltas computed: count per session
   • Full resyncs: when and why
   • Bandwidth saved: total bytes avoided
   • Average delta size: per-update efficiency

4. THREAD-SAFE
   • Snapshots are read-only
   • No locks needed for computation
   • Connection send already mutex-protected

5. ERROR RECOVERY
   • Graceful fallback on any error
   • No risk of desynchronization
   • Detailed error logging

API REFERENCE
================================================================================

High-Level (Recommended):

  from addon.sync import initialize_scene_tracker, compute_scene_delta

  # After full scene sync:
  initialize_scene_tracker(bpy.context)

  # On viewport updates:
  delta = compute_scene_delta(bpy.context)
  if delta and not delta.full_resync_required:
      conn.send(MsgType.SCENE_DELTA, delta.to_dict())

  # Monitor:
  metrics = get_delta_metrics()

Low-Level (Advanced):

  from addon.scene_diff import SceneDiffer, SceneDiffEngine

  snap1 = SceneDiffer.snapshot_scene(context)
  # ... user modifies scene ...
  snap2 = SceneDiffer.snapshot_scene(context)
  delta = SceneDiffEngine.compute_delta(snap1, snap2, context)

See DELTA_SYNC.md for complete API documentation.

================================================================================
KNOWN LIMITATIONS
================================================================================

Geometry edits → Full resync required
  Mesh editing requires full .blend context
  Future: Geometry streaming protocol

Material edits → Full resync required
  Shader node networks are complex
  Future: Node-level diffing

New objects → Full resync required
  Need full scene structure context
  Future: Lightweight object creation protocol

Texture paths → Assumed shared
  Delta doesn't sync texture files
  Workaround: Pack textures or use shared folders

All limitations are gracefully handled with automatic fallback.
