WINDOWS SERVER CHANGES
======================
server/renderer.py
  + Added json import
  + Updated render_final() signature:
    - Added denoiser_type parameter (default="OFF")
    - Added denoiser_intensity parameter (default=0.8)
    - Added auto_aov_creation parameter (default=True)
    - Passes parameters to _build_render_script()
  + Updated _build_render_script() signature and implementation:
    - Added denoiser_type, denoiser_intensity, auto_aov_creation parameters
    - Generates OptiX denoising config when denoiser_type="OPTIX"
    - Generates OIDN denoising config when denoiser_type="OIDN"
    - Creates Normal and Diffuse Color AOVs when auto_aov_creation=True
    - Sets render.use_denoising = True
    - Sets render.denoiser = 'OPTIX' or 'OPENIMAGEDENOISE'
    - Includes denoising-specific logging in generated script

server/denoiser.py (NEW FILE)
  + Created DenoiserManager class
    - _check_optix_available(): Detects NVIDIA OptiX support
    - _check_oidn_available(): Confirms OIDN availability
    - get_available_denoisers(): Returns ["OFF", "OIDN", "OPTIX"] (subset)
    - denoise_exr(): Post-process denoising framework
    - log_denoising_result(): Logs denoising performance metrics

server/server.py
  + Added import: from server.denoiser import DenoiserManager
  + Updated RenderServer.__init__():
    - Added self.denoiser_manager = DenoiserManager()
    - Added self._render_settings dict with defaults:
      - "denoiser_type": "OFF"
      - "intensity": 0.8
      - "auto_aov_creation": True
  + Added _handle_render_settings() method:
    - Receives RENDER_SETTINGS message from client
    - Validates denoiser_type (OFF, OIDN, OPTIX)
    - Clamps intensity to [0.0, 1.0]
    - Falls back OPTIX → OIDN if OptiX unavailable
    - Stores settings in self._render_settings
    - Logs: "[session_id] Render settings updated: OPTIX denoising (intensity=0.8)"
  + Updated _handle_message() router:
    - Added elif msg_type == MsgType.RENDER_SETTINGS branch
  + Updated _handle_render_start():
    - Passes stored denoiser settings to final_renderer.render_final()
    - Calls denoiser_manager.log_denoising_result() after render
    - Logs format: "OptiX denoising applied (intensity=0.8, took 234ms)"
